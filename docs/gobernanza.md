# Gobernanza de Datos

Este documento detalla las reglas y estándares aplicados durante la gestión del pipeline de datos.

## Estándares
- Esquemas bien definidos y documentados
- Control de versiones de datos (por capas: raw, bronze, silver)
- Logs y trazabilidad de errores

## Buenas prácticas
- No modificar datos en bruto
- Toda transformación debe ser reproducible
